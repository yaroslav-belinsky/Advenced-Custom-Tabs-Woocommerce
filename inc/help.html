Ninja Forms Data Redirect

For —Åorrect configuation of sending data from Ninja forms to API responder.co.il you need to follow these requirements:

  1. Enter to the page Ninja Forms Data Redirect -> Forms Settings and choose API settings

    2. On this page, you must set API url, example "https://cp.responder.co.il/subscribe.php".

    Also you can enable Debug mode, what can help you to check the form. Attention! When Debug mode is turned on, we call method wp_die() to get information.

  2. Go to the page Ninja Forms Data Redirect -> Forms Settings and select tab Ninja forms

    3. On this page we need to connect the form to API. Click to the button "+" and you will create new connection. You need to enter these fields in parametres:

    • Ninja form ID - this is ID of Ninja form
    • ID API - this is ID of connected form in responder.co.il
    • Link Thank you - this is address where you are redirected after successful submission of the form
  3. Go to editing of Ninja form which you need

    4. For connection with API you need these fields:

    • subscribers_name
    • subscribers_email
    • subscribers_phone

    Plugin need to get this fields from Ninja forms you need to configure form:

    • Open configuration of field for example "First name"
    • Open - "Advenced Settings" of this field
    • Put to - "Admin Label", for "First name" it would be "subscribers_name")
    • Put to - "Custom CSS Classes",you need to put same thing as about but you need it for validation of form (for "First name" it would be "subscribers_name")
    • if you want to make this field required in "Custom CSS Classes" you should add "required", so this field "Custom CSS Classes" for "First name" would look like: "subscribers_name required". You need put SPACE between it.

Congratulations! You have configured the plugin.
